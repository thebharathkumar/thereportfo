# Resume Files

This folder contains resume PDFs for different job roles.

## Files to Add

Please add the following PDF files to this directory:

1. **software-engineer-resume.pdf** - General software engineering positions
2. **data-engineer-resume.pdf** - Data engineering and analytics roles
3. **ml-engineer-resume.pdf** - Machine learning and AI engineering positions

## Usage

These resume files are referenced in the portfolio website's download modal. Users can select and download the resume type most relevant to the position they're viewing.

## Note

Make sure to:
- Keep all resume PDFs updated with your latest experience
- Use consistent formatting across all resume versions
- File size should be optimized for web (ideally < 1MB per PDF)
- Include your contact information on each resume
