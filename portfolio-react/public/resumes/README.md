# Resume PDFs

Place your resume PDF files in this folder with these exact filenames:

- `software-engineer-resume.pdf` - For general software engineering positions
- `data-engineer-resume.pdf` - For data engineering roles
- `ml-engineer-resume.pdf` - For ML/AI engineering positions

## How to Add Resumes

1. Export your resume as PDF from your preferred tool (Google Docs, Word, LaTeX, etc.)
2. Rename the file to match one of the names above
3. Place it in this folder (`/public/resumes/`)
4. The Download Resume button will automatically link to these files

## File Requirements

- **Format**: PDF only
- **Size**: Keep under 2MB for fast loading
- **Name**: Must match exactly (case-sensitive)

## Testing

After adding your resumes, click the "Download Resume" button in the Hero section to test the download functionality.

The resume modal will show all three options, and users can choose which version to download.
